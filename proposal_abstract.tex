\begin{tcolorbox}[abstractstyle,
  title={执行摘要 \quad {\normalsize\normalfont\itshape (Executive Summary)}}]

纠错码（Error-Correcting Code, ECC）是贯穿现代信息社会全域的核心基础技术——无论是5G/6G无线通信、相干光纤传输、NAND Flash固态存储、量子计算系统，还是新型存算一体器件，无一例外地高度依赖纠错码保障信息可靠传输与处理。然而，尽管应用场景千差万别，当前ECC设计范式却被三大结构性困境所困：\textbf{（一）过度依赖专家经验}——从码型构造、度分布优化到译码算法选择，全程仰赖跨越信息论、信号处理与VLSI设计的深厚专家知识壁垒；\textbf{（二）优化严重依赖大规模仿真}——获取一条可靠的BER性能曲线须耗费数小时乃至数天的蒙特卡洛仿真，使大规模码型搜索在工程上几乎不可行；\textbf{（三）算法优化与硬件实现孤立割裂}——译码算法与芯片硬件设计两条并行轨道互不知情，导致算法确定后硬件端大量推倒重来，严重拖慢标准化进程。

\vspace{4pt}
本提案提出 \textbf{PrimeCoDec}——一个面向信息系统纠错码需求的\textbf{全链路AI自动化框架}（Full-Pipeline AI Automation for Code Design and Decoding）。PrimeCoDec 将上述三大困境同时消解：用户仅需以自然语言输入原始需求，框架即可\textbf{全自动生成}从纠错码编码方案、译码算法、硬件部署方案直至行业标准草案的完整输出，全程无需专家干预。

\vspace{4pt}
框架的核心创新体现于四个维度：\textbf{（一）通用信道三元组抽象}，将任意物理系统噪声统一表征为$\calT{=}(\{\SNR_i\},\bvec{R},p(x))$，赋予框架天然跨域泛化能力；\textbf{（二）神经算子性能预测}，以DeepONet在毫秒级预测完整BER/FER曲线，相较蒙特卡洛仿真实现逾$10^3$倍加速，根本上破除仿真瓶颈；\textbf{（三）算法--硬件联合代理优化}，在统一目标函数下同步优化算法性能与VLSI指标，终结``先算法后硬件''的串行模式；\textbf{（四）LLM驱动的需求解析与标准生成}，将自然语言规范自动转化为形式化约束，并将优化结果反向输出为符合3GPP/IEEE格式的行业标准草案。

\vspace{4pt}
作为核心验证案例，我们以\textbf{5G NR标准（3GPP Release 17）}为基准对比——这是当前工业界极少数经数十年专家深度打磨、充分优化的行业标准之一，因此该对比天然公平且极具说服力。PrimeCoDec 全自动生成了\textbf{性能优于5G NR的Polar码与LDPC码}，给出当前场景下的最优译码算法，并设计了\textbf{共模架构译码器}（一套硬件同时实现Polar码与LDPC码译码），最终输出了修订后的3GPP编码标准草案——从用户输入到可部署标准，全程由框架自动完成。

\vspace{4pt}
预期PrimeCoDec将ECC设计周期从月/年级压缩至\textbf{小时级}，首次实现面向非专家用户的民主化ECC设计，并在无线通信、光通信、数据存储、量子计算等多个战略领域具有广泛的深远影响。
